# Supporting materials for bachelor's thesis "Accessibility of Mobile Retail Applications for Users with Vision Loss:Audit, Recommendation Framework, and AI Support for Accessibility Analysis"

This repository contains following files:
* Audit_[name of the app].xlsx - 4 strucuted tables of issues detected durnig the automatic, usability, and manual testings
* Usability_testing_scenario.ms - scenario that was used during usability testing. Contains structure of the meeting, tasks, survey
* Accessibility_issue_writer.json - workflow of AI assistant impoted from n8n
* Usability_scores.xlsx - strucutred usability survey scores results
* Session recordings were collected with participants’ consent and are retained as evidence of the conducted usability testing sessions. However, because the recordings contain personal information, they are not publicly shared as part of this repository. Access to the recording may be granted upon separate request
